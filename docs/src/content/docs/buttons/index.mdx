Reusable button components provide common CRUD and UI actions with consistent styling, i18n, and side effects (navigation, mutations, list refresh, etc.).

Common patterns:
- Most accept a `label` prop (string -> translated) or derive a default i18n key (e.g. `ra.action.create`).
- Buttons used inside list or table rows call `stopPropagation` to avoid triggering row click.
- Destructive operations use undoable or optimistic mutations via reactâ€‘admin hooks.
