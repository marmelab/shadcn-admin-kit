---
title: "Spinner"
---

Simple animated loader with size variants.

## Usage

```tsx
<Spinner size="large" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"small"\|"medium"\|"large"` | `medium` | Visual size |
| `show` | `boolean` | `true` | Toggle visibility |
| `className` | `string` | - | Extra classes |
