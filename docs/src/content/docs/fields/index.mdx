---
title: "Fields"
---

Field components display values from a record context. They are typically used inside `<DataTable>`, `<Show>`, or custom layouts.

Common props:

- `source`: path to the value in the record
- `record`: override the current record
- `defaultValue`: fallback when value missing
- `empty`: placeholder (translated if string)

Available field components:

- [ArrayField](./ArrayField.md)
- [BadgeField](./BadgeField.md)
- [DateField](./DateField.md)
- [EmailField](./EmailField.md)
- [FileField](./FileField.md)
- [NumberField](./NumberField.md)
- [ReferenceArrayField](./ReferenceArrayField.md)
- [ReferenceField](./ReferenceField.md)
- [ReferenceManyField](./ReferenceManyField.md)
- [SelectField](./SelectField.md)
- [TextField](./TextField.md)
- [UrlField](./UrlField.md)

For input components (editing data) refer to [the Inputs section](../inputs).
