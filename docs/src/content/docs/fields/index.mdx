---
title: "Fields"
---

Field components display values from a record context. They are typically used inside list, show, or custom layouts. All fields accept a common subset of props from `FieldProps`:

Common props:

- `source`: path to the value in the record
- `record`: override the current record
- `defaultValue`: fallback when value missing
- `empty`: placeholder (translated if string)
